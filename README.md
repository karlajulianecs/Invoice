##  Invoice

Crie uma classe chamada Invoice, que possa ser utilizado por uma loja de suprimentos de informática para representar uma fatura de um item vendido na loja

Os sub-tópico abaixo descrevem algumas decisões tomada na estruturação utilizada no projeto.


Uma fatura deve incluir as seguintes informações como atributos:

• Número do item faturado

• A descrição do item

• A quantidade comprada do item

• O preço unitário do item
